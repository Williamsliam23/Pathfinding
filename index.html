A* Pathfinding Visualization

About

A* is a graph traversal and path searching algorithm that was developed in the 1960's. The algorithm is created in a best first search way, weighing the movement of traversal from the beginning node to the ending node following a f(n) = g(n) + h(n) formula; n being the next node. 'g' is the constant cost for movement in this example being used as 1 for 2-D and non diagonal traversal, but can be varied with diagonal and 3-D movement.
